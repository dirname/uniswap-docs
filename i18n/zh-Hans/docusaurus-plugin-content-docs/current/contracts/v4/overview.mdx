# Uniswap V4 - Hooks

欢迎来到 Uniswap V4 开发者文档。

Uniswap V4 设计用于增强链上提供流动性和交易代币的方式，旨在拓宽去中心化交易的范围。该项目欢迎社区参与，为开发者和爱好者提供了一个平台，让他们能够为这个 Uniswap 协议的新版本做出贡献。

Uniswap V4 的一个显著特点是引入了“Hooks”。这些合约在池操作生命周期的不同阶段运行，允许进行高度定制。

池创建者可以选择保留 V3 中可用的权衡决策，或者探索新功能。Uniswap V4 池可以支持动态费用、链上限价订单，或者作为时间加权平均市场做市商（TWAMM），将大额订单分散在一段时间内执行。架构已经更新，包括一个新的“单一”合约，其中所有池都包含在一个智能合约中，提高了平台的效率并降低了成本。

Hooks 与单一架构的集成预计会使平台更加强大和多功能，确保在各种池之间快速、安全、高效地进行定制和路由。

注意：V4 目前正在开发中，合约中的更改正在进行。文档将相应地进行更新。
