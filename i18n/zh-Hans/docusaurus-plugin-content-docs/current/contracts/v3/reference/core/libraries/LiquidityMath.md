## 函数

### addDelta

```solidity
  function addDelta(
    uint128 x,
    int128 y
  ) internal pure returns (uint128 z)
```

向流动性添加一个带符号的增量，并在溢出或下溢时回退。

#### 参数：

| 名称 | 类型    | 描述                                    |
| :--- | :------ | :--------------------------------------------- |
| `x`  | uint128 | 变更前的流动性                    |
| `y`  | int128  | 应该改变流动性的增量 |

#### 返回值：

| 名称 | 类型    | 描述         |
| :--- | :------ | :------------------ |
| `z`  | uint128 | 流动性增量 |
