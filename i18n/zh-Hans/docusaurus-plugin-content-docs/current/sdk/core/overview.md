# Uniswap 核心 SDK

> **欢迎来到 V3 核心 SDK！**

Uniswap 核心 SDK 在 Typescript/Javascript 环境中为其他 SDK 提供了抽象层。在整个 Uniswap SDK 中都有所应用，例如 [V3 SDK](../v3/overview.md)。

此 SDK 不期望单独使用，而只作为其他 SDK 的一部分。

我们建议您查看 [**技术参考**](./reference/overview)。
