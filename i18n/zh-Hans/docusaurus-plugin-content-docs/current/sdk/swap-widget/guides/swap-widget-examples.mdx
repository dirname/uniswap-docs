# 交换小部件示例主题

[入门指南](/sdk/swap-widget/guides/getting-started)展示了如何嵌入交换小部件，让你的用户在不离开你的 dApp 的情况下，在 Uniswap 协议上交易代币。

下面你会找到一些示例，展示如何[自定义小部件主题](/sdk/swap-widget/guides/examples#customizing-theme)，以匹配你的 dApp 的外观和感觉。所有这些都可以通过以下代码片段进行集成，其中你可以设置你的 `theme`：

```tsx
import { Theme, SwapWidget } from '@uniswap/widgets'
import '@uniswap/widgets/dist/fonts.css'

const theme: Theme = {
  // 查看下面的主题示例
}

function App() {
  <div className="Uniswap">
    <SwapWidget theme={theme} />
  </div>
}
```
